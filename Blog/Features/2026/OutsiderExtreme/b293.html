Project Outsider X-Treme

X-Treme Modding

Barely a day after Retro Randy Price's WCW Born to Wired mod has been released, G.M.Spectre has dropped the patch for his on gong WWF No Mercy mod; Project Outsider X-Treme.

This mod, which has been seen various iterations since its inception in 2022, features a weird and wonderful roster of whacky faces as well as wrestlers. Check out the fever dream-like footage below:

X-Treme Rosters

Project Outsider X-Treme has adopted a lot of changes since the last "Plus" version that was released in 2024. This is due to being built off the back of Retro Randy Price's No Mercy Plus v3 mod, which allows for expanded rosters without an external save file, plus save game fixes and more!

 

Latest Changes:

Full roster is set be default, no save file needed.

No Mercy save bug is fixed, saves will not be erased.

Expanded roster, 108 characters in total.

Entire roster can be completely edited.

Tag team entrances and factions added.

Breakable tables, Hold C-up when getting a weapon from the crowd.

Roster expanded and slightly revamped.

Revamped groups.

More costumes added.

Many more CAW outfit faces and clothes added.

Smackdown Mall set with unlockable characters, attires, and other things.

Much more music added.

Patching your ROM is super simple; just obtain the WWF No Mercy (U) [!] Z64 ROM with an MD5 of 04C492BE7F89FC6F425238BD67629544 and using either the online ROM patching site or the Aki Guide Patcher (linked below), modify your ROM in seconds!

In order to play this mod on SummerCart64, you will need to do some extra steps to ensure your flash cart is on the correct firmware, because 2.18.0 is the only version that currently fully works with this mod. Analogue 3D users should also ensure their FPGA systems are updated to 1.20 or higher.
