VPW2 Recomp WIP

Recompiled Wrestlers

Fresh news this morning via Grix suggests that a PC recompilation of AKI Corporation and Asmik Ace Entertainment's Virtual Pro Wrestling 2 is in active development, with features such as widescreen mode, modern controls and enhanced graphics and framerates.

The project is still in very early development, but taps into the VPW2 decompilation efforts put forward by Aki-Club, a group of talented modders and wrestling fans who include bulletgospel, gumorr, MelonBreadVPW and Freem of VPW2 Freem Edition fame.

Kings of the Ring

While the project is in its early stages, this will no doubt fire up the N64 wrestling community, of which there is a sizeable following. With notorious sceners such as Zoinkity and WldFb having contributed to ROM hacking via documenting and editing as far back as 2008, we should hopefully also see some of the excellent hacks and mods arrive for the source ports when they are eventually made available.

Expect me to cover updates on this recomp as it progresses, because the N64 wresting era offers a heavy-weight dose of nostalgia, even for non-wrestling fans such as myself. I played these games a lot without knowing the characters or the stories, mainly for the outlandish laugh-out-loud moments these games offered and the bragging rights when you mash up your mates with bats, tables and chairs!

Virtual Pro Wrestling 2 RECOMP

================================

Features:

Native Windows executable

RT64-powered rendering

Widescreen support

Borderless fullscreen launcher

Improved modern resolution output

Keyboard and XInput controller support

Editable input.ini control mapping

Discord Rich Presence

Free camera mode with F8

Save/config folder support through vpw2_userdata

================================

This package does not include a ROM or any copyrighted game assets.

Quick start:

1. Put your legally obtained Virtual Pro Wrestling 2 - Oudou Keishou (Japan) ROM in this folder.

2. Use one of these filenames:

- Virtual Pro Wrestling 2 - Oudou Keishou (Japan).z64

- vpw2.n64.jp.1.0.z64

- rom.z64

3. Double-click "Launch VPW2.bat".

"Launch VPW2.bat" starts the game in borderless fullscreen with widescreen enabled.

"Launch VPW2 Windowed.bat" starts the game in a normal window.

Expected ROM:

Virtual Pro Wrestling 2 - Oudou Keishou (Japan)

SHA1: 82dd25a044689eab57ab362fe10c0da6388c217a

Controls:

The game creates vpw2_userdata/input.ini on first run. Edit that file to remap keyboard

or XInput controller bindings.

Notes:

- Saves and config are stored in vpw2_userdata.

- Press F8 during a match to toggle free camera mode.

- Discord Rich Presence is enabled when Discord is running.
