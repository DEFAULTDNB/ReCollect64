Swap, Dump, Swap, Dump

Did you know that you can dump N64 ROMs directly from a cartridge to a flash cart, without any need for any specialist equipment or even a PC involved?

Mntorankusu has recently released an updated version of their homebrew ROM dumping solution named N64-SwapDumper, an application that, once loaded, uses a hot-swap method to load a portion of an inserted cartridge into RAM, then dumps it back to the flash cart once re-inserted.

While the method works, it does come with risks for both your hardware and software (including the flash cart), so please check out the information below:

 

N64 SwapDumper

Cartridge-swap-based cartridge dumper for Nintendo 64, requiring only a flash cart. You can dump saves, ROM, or both. Inspired by and based on sharksaver64 by @Jhynjhiruu @ppcasm @Modman

First Release (v0.1 / Baby SharkSaver) Pre-release

First release. Based on sharksaver64 by @Jhynjhiruu @ppcasm @Modman

Only tested on SummerCart. If you dare to try it, please report any issues you find. To use, run the application on your flash cart and follow the on-screen prompts. While there are two dumping modes available, the hardware for the accessory slot dumping mode does not yet exist, so you must choose SD card mode.

Dumping games with this application involves swapping cartridges while the console is powered on. Only use this if you accept that you may damage your console. No warranty is provided. Incremental Improvements (v0.2 / Shark Week) Latest

This version has some small but important improvements:

+ The cartridge info screen now shows more info, including revision info and the ROM header checksum, so the tool can double as a cartridge identifier even if you are not planning on dumping anything. Different revisions of the same game will receive different filenames now, too, instead of overwriting each other.

+ A full CRC32 checksum is calculated before and after dumping to make sure that the ROM on your SD card matches the cartridge exactly.

+ If sd:/dump/ and/or sd:/saves/ exist, files will go there now. Otherwise the dumper falls back to the SD root. Save dumping now also warns before overwriting an existing save.

+ Compression speed has improved while keeping the compression ratio essentially the same.

+ The ROM size selector now includes 40 MB as an option. Not sure if any other ROM sizes are still missing.

ROM-ifications

Interestingly, this method reminds me of the Stop 'N' Swap mechanic Rare wanted to implement in Banjo-Tooie. The idea was that while the game was running and loaded into RAM, you could pull out the game cart, shove in another, and unlock secret characters and items. Nintendo famously nixed this idea partly due to the unnecessary wear and tear put on the console's components while powered on, but also because the Nintendo 64 was going through a process of getting new revisions, which made the amount of time anything was stored in RAM clear considerably quicker.

I personally have not tested this method of dumping, because I don't wish to damage my consoles, carts, or flash carts (of which all major carts are supported), and also because I have already dumped my games using the JoeyN64 and an off-brand clone dumper.

It would certainly be interesting to know if newer revision consoles were also capable of dumping games. If you happen to use it, please let us know!
