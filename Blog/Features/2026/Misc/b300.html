N64 Apple Watch Ports?!

Exceptional Experimentation

Two months ago, I noticed that someone managed to get Super Mario 64 running within a PDF, yes, a Portable Document Format file, through the magic of JavaScript and some intricate coding tomfoolery. Sadly, I wasn't able to cover that story at the time; however, with the latest advancements and experiments from that very same developer, Game of Tobi has done the unthinkable and gotten decompiled Nintendo 64 games up and running on an Apple Watch!

It should be noted that at the same time as the PDF insanity, Game of Tobi also ported Super Mario 64 to the fruit company's wearable, something I never thought would be done in a million years. Regardless, check them out in action below:

Decompilation Derivation

Thanks to decompilation projects for each game, it's now possible to recompile them for almost any platform; however, critically, Apple Watch doesn't support SDL2, so some effort was made to rewrite everything in SpriteKit, which leverages Metal to achieve high-performance 2D rendering on Apple devices, and eventually SceneKit, which is better suited towards 3D content rendering.

The ports leverage a similar method to that of PC recompilations, which requires a legally obtained ROM, its assets dumped, and in this case, an Xcode framework to run within. Each game can be started up, a save game made, and technically, they can be played to completion, but the overlays for the buttons do take up a lot of valuable real estate on that minuscule screen, so it looks a little fiddly to play.

Tobi wishes to crush bugs, increase performance and make it a more rounded experience before releasing it all publicly, and there are plans for more game yet, but I have to say that while it might not be the ideal way to play portably (on barely an inch-wide screen) it's a mightily impressive technical proof of concept and are smashing additions to the ever-increasing N64 port library nonetheless!
