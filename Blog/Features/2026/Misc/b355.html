DK64: Recomp Announced

Not Monkeying Around

A fresh recompilation project has been announced today by way of 2dos on YouTube. The project which involves Rainchus, Ballaam, Killkili and Umed is none other than the long-awaited PC, macOS and Linux port of Donkey Kong 64, which utilises the incredible N64: Recompiled toolset and the ongoing DK64 decompilation.

The recompilation already touts support for quality-of-life updates such as high frame rates and low input lag, as well as mod support for key ROM hacks such as Isotarge's "tag anywhere" mod, which allows you to change Kongs at any point in the game, which prevents backtracking and gives the game a somewhat better flow.

Banana Bunch

It should be noted that a complete 100% decompilation is not required to make a recompilation. In essence, what you need is good disassembly, manually identified Libultra functions, and several patches to get the game going. Having access to a complete and labelled codebase is always going to be handy, but it isn't strictly necessary. The DK64 Randomiser team's involvement could be incredible here, because their current online ROM hacking tools allow for complete randomisation of items, locations, win conditions, and much, much more!

With Rainchus' involvement with the DK64 randomiser team, I have a good feeling that big things will come of this port and that we will see a nice, human-led recompilation to add to the ever-increasing number of N64 source ports we have access to. As far as I know so far, no AI assistance has been used in this project, but we will have to wait for the release to verify the code.

Rainchus has previously had a hand in the Quest 64 recompilation, several unreleased recomp projects (MP3, Chameleon Twist, Glover, and Mario Golf 64), and the rather awesome Mario Party 3 ROM hack Mario Party Combination, which was released last year.
