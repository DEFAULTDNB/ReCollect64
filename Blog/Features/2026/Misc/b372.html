Portable Pugilists

Yes, you're reading that right; Super Smash Bros. for the Nintendo 64 is currently in the process of being ported to the Nintendo DS console - no, not the 3DS or Nintendo Switch, the OG 2004 handheld. The project is being spearheaded by Rockenrooster, who is employing the use of JRickey's BattleShip, the AI-assisted PC, Mac, and Linux source port of the Nintendo 64's 1999 fighting title.

 

This project combines the original game logic reconstructed by the BattleShip project with a Nintendo DS backend. The result is a playable Nintendo DS port rather than a new Smash-style game written from scratch.

The current build focuses on a one-minute Mario-versus-Fox match on Dream Land, with items off. Bring your own legally obtained North American Super Smash Bros. ROM; this repository contains no Nintendo assets or ROM-derived data.


Been working on porting (codex assisted) n64 super smash bros to the Nintendo DS for the past month. I knew nothing about coding for the DS before this, but i do have some C# and Powershell/python knowledge.

Here is what I have to far:

Mario and Fox on the Dreamland stage running about half speed (half speed is a stretch since the original game ran at 60fps and we're getting like 15 fps here in gameplay, i had to change the game logic (2 updates per frame) so it didn't run at 1/4 speed lol).

Still have a long way to go, but i'm pausing progress for now since GPT5.6 sol is struggling (ended my $200 codex sub back to $20 for now).

AI Attack

While the demo is extremely early, massively limited, barely runs between 12 and 19 FPS, and unreleased to the general populace; there is a husk of possibility here, that an effigy of the seminal fighter might actually get to run on the discontinued and multifariously replaced handheld.

The chaining together of an AI-based recompilation to yet more AI assistants to craft what is essentially a demake-style port is a peculiarly layered use case. I just wonder how much of this code is human-crafted versus how much is AI-generated. It will be interesting to observe how far this one goes.
