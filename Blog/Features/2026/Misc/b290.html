BK on Rare Super Pocket!

Retro-Portability

Rare have today announced that they have partnered with Hyper Mega Tech to produce a limited edition Super Pocket which has not one, but 14 games built in, including the Nintendo 64's Banjo-Kazooie.

Critically, the N64 game is built into the limited edition console, meaning that you cannot buy it separately yet. Purchasing the console will set you back just £49.99/$59.99/€69.99 but will only include one 64-bit-era game to enjoy on the move.

Retooled Rarity

This particular edition of Banjo-Kazooie has been optimised to not only work with a D-pad, but also work with the 8-to-32-bit cartridge-based portable. This marks the first actual 64-bit game to come to the Super Pocket, or any other Evercade compatible devices, because usually it's the PSX ports of recognisable N64 titles, such as Glover or 40 Winks (which was cancelled before release on the N64), that make the cut.

Banjo-Kazooie is currently playable on Nintendo Switch Online + Expansion Pack, Xbox via the 2015 Rare Replay re-releases, and most recently on PC and via recompilation. The recompilation, released in January, was the first time that the game was made portable, unofficially, as it came with a Flatpak Linux release which is fully compatible with Steam Deck.

It would be interesting to see this edition of the platformer dumped and analysed to see what else has changed and what, if any, other N64 games we might be able to get running on the pocketable hardware. Chances are this is a direct port rather than emulation, so I wonder if the decomp has been tapped into at all.
