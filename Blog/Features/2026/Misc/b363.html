UPDATED WonderMenu WIP

A Wondrous Flashcart GUI ~ Jul 14, 2026

Modern Menu Marvels

Homebrew developer Lee McDermott (LMCD) has been showcasing his incredible-looking WonderMenu flashcart menu for the past two weeks, but I have held off posting until now because there were no signs of a possible release date. Today, that changes, because it could be right around the corner with the dev stating, "Full release most likely coming this week".

Notable features include 3D carts with full label art and category tabs for homebrew, history and favourites. Each game immediately displays information such as the region, the developer, and whether or not it's been added to favourites, and a sub-menu for each game then shows available cheats that you can toggle on or off, as well as viewing in-game screenshots and editable per-game settings.

It should be noted that the screenshot implementation in the pre-release videos shows images read from a folder on the SD card, but Lee M has been looking into the feasibility of dumping the frame buffer mid-game to achieve the taking of actual screenshots. According to his comments, he actually might have a 320 x 237 pixel bitmap capture method ready and implemented, which only works on Super Mario 64 and Mario Kart 64 for now via a "90s GameShark code" implementation, and the press of a simple controller button combo. We will have to wait a little longer before we can see it in action and to see if more games can be configured to dump screenshots within the WonderMenu's database.

The latest video posted of pre-release version 0.3 also demonstrates the ability to playback TAS (tool assisted speedrun) files directly on your console, with the added memmory of an Expansion Pak, which is something I have never seen before. From the looks of this menu, the sky is the limit for innovative features and exquisite usability!

Flowing With Features

Based on the N64FlashcartMenu and with a stunning UI that houses a wonderfully clean display of cartridges, I wonder if Lee M will implement features such as additional box art backgrounds or even simple tweaks such as editable cartridge colours (in much the same way Analogue did in the 3D's 1.2.6 update).

All that remains is for a release date to be set in stone, which the developer has stated should be extremely soon, as this is the last pre-release video for the project. Hopefully we should see a release in the coming days, and I'll be sure to update this post the second it lands.

I for one will be delving deeply into this menu when it's available because it's visually solid and appears to feature a lot of things I would appreciate in a flashcart menu. Screenshots alone would be revolutionary, as it's not been done before, and when I previously asked EverDrive developer Krikkz about adding that feature to the EverDrive-64 X-series, he simply replied something to the effect of, "Don't hold your breath". Perhaps this new implementation will eventually sway more devs to include a screenshot feature, or spur on the progress of the N64FlashcartMenu being ported to ED64 X-series carts.

So I’ve been obsessing over the N64 for years. This year I got an Analogue3D, and finally had some time to make something special!

WonderMenu is a flash cart menu system for the N64, with some quite juicy features!

+ Beautiful/responsive interface running at 480i, 60fps

+ Extra visual enhancements on Analogue 3D with 32bit colour and progressive output

+ Comprehensive retail games database with high resolution artwork and metadata

+ Cheats database for hundreds of games

+ Ability to capture screenshots mid-game

+ Ability to playback tool assisted speedrun files from TASVideos.org

+ Games of different regions/versions are grouped together

+ Works on SummerCart64 (other carts to come!)

+ Imports your recents and favorites from N64FlashCartMenu

Pre-release builds, and builds with extra features like speedrunning and screenshots will be made available on Patreon! There will be a free version supporting most features open sourced on GitHub - I'll get around to this in the next day or two.

This is a real passion project of mine, and I can barely afford to keep going on this with the current hours I’m putting in. Please support WonderMenu on Patreon - I can’t keep going without it. With some support, I hope to deliver the following:

+ Save state support

+ Record your own speedruns

+ On screen timer/UI for speedruns

+ More high resolution game artwork

+ Themes/customisation

+ Expanded flash cart support

+ More nice UI improvements/animations

Some caveats on 0.4:

+ This is a BETA pre-release build with some work to go

+ Screenshot capture is implemented for a few games, but the UI isn't implemented yet - I want to get it right if it's going to render at 60fps

+ Speedruns are enabled for MK64-Europe and SM64-US. Drop .m64 files into /speedruns

+ Button indicators aren't added yet. Z button to favorite, and Z button to show cheats screen/tabs

+ Only currently reads roms from top level directory

+ Not all cheats have been verified to work. In later releases, I'll be marking working cheats as 'Verified'

+ Only SummerCart64 is verified working for now. Other carts to follow
