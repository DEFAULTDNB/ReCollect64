Rock 'n' Roll

Rinnegatamante is on a roll with this source ports to PS Vita, this time we have none other than Starship, aka Star Fox 64, running natively in the first ever release for the PS Vita!

The port features all the usual mod cons you would associate with his fantastic ports, including native performance, custom resolution/graphic options, remappable controls, cheat menu readiness, and high framerates.

Description:

Starship is a sourceport of StarFox 64 based off the decompilation efforts of the game. It features several enhancements and mods support. The Vita port is based on version 1.0.0 (Centauri Alpha) with bugfixes from the in-development branch.

Requirements:

- libshacccg.suprx

- Game Data Files: N64
