Harvest Moon 64: Recomp WIP

Happy Harvesting

Back in December I reported that the Harvest Moon 64 decompilation was complete, and this week, around four months later, we have news via OpenSource on YouTube, that the recompilation effort is well underway and that we could be enjoying it sooner rather than later!

 

Play Harvest Moon 64 natively on PC through static recompilation. Coming Soon!

Includes properly scaled UI for ultrawide displays and day-one mod and texture support through N64Recomp.

The project does not include game assets. The original game is required to play.

N64Recomp developed by Mr-Wiseguy

Farming Features

Demonstrating ultrawide screen modes, scaled UIs, refined frame rates and higher resolution graphics, this project uses Mr Wiseguy's N64: Recompiled toolset as a base, so you know it's heading in the right direction for a static recompilation, and that it will have proper mod and texture support from day one.

For now there is no set release date, so keep you eyes peeled for the release, and then, hopefully the ports to other consoles which might follow!
