Donut Dodo Coming to N64

Physical Pixels

It's not every day that the Nintendo 64 gets a new game physically released, so when I saw that a new cartridge-based game was heading towards my favourite console, it goes without saying that I was elated.

Donut Dodo, by Luxembourg-based Pixel Games Sarl, is a retro-inspired 2D arcade game, which sees Baker Billy Burns hopping, skipping and jumping around the screen, avoiding the titular Dodo's projectiles and attacks as well as collecting donuts as quickly as possible.

 

Donut Dodo is a fast-paced arcade platformer inspired by the golden age of coin-op classics from Namco and Nintendo.

What started as a personal love letter to those early games gradually evolved into a fully fledged arcade-style experience for modern systems.

For the Nintendo 64 version, the game was rebuilt from scratch in C to run on original hardware, staying true to the spirit of classic “home conversions,” while adding new modes and features.

It’s about creating the kind of game that once sparked the desire to become a developer in the first place, now coming home to the systems that inspired it.
