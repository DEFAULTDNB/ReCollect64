Testing on the Go! ~ Aug 27, 2024
N64 Pad Test Rig

Portable Perfection

On August 26th, @RockerGameMod and @drumstix576 presented their latest retro gaming project that should help N64 players verify their controllers on the move. The portable pad testing rig allows you to plug in a controller and verify the sticks range of abilities against a custom piece of software akin to Mimi2's Stick Tester ROM for use via N64 flash carts.

The creation of this device has been a passion project of RockerGaming's, and it means that their dream of testing Nintendo 64 controllers on the go is now a reality!

RockerGaming is well known for their Renaissance Bowl Steel Bowl mods for N64 controllers, controller switchers, carbon fibre stick caps, and trophy-style controller display stands via their Etsy store.

This Box Rocks

Upon contacting RockerGaming for some more information, they shared that the project uses a custom PCB with an RPi Pico as its microcontroller. The code used is entirely custom and has been built from the ground up too. It reads controller inputs and displays their values, cleanly indicating any deviation, on the built-in screen.

With its sturdy plastic casing and simple USB-C charging for the lithium-ion battery, this project looks to be incredibly well rounded and close to production.

What other features may be included remains to be seen, but this project is very exciting and extremely useful. It goes without saying that I'm very impressed, and I definitely look forward to checking in on this project in the future!
