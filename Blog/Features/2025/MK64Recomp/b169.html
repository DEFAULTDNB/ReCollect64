MK64 Recompiled Released

Recompiled Racing

This morning SonicDcer shared a progress update from his Mario Kart 64 Recompiled project. Not to be confused with Spaghetti Kart, which is the PC port built from the recently updated, byte-perfect Mario Kart 64 decompilation, MK64: Recompiled uses the static recompilation methods set out by Mr-Wiseguy and the RT64 renderer from Dario.

 

Features:

Mario Kart 64 - Recompiled is a project I started back in January of this year, it's still a work in progress but I already made it public for anyone to try it out.

The game is fully playable and stable.


Regarding Texture Packs:

Enable developer mode in your %appdata%/MarioKart64Recompiled/graphics.json

Press F1 inside the game to bring up the RT64 debugger.

Textures->Load directory and select the texture pack folder.

I must warn you it's still a WIP.

To run the recomp, you will need the USA *.Z64 ROM (MD5: 3A67D9986F54EB282924FCA4CD5F6DFF) and the files from the GitHub repo below. You can then download Ghostly Dark's HD texture pack and play the game with the crispest textures and high frame rates, plus widescreen and ultrawide support, customisable controls and deadzones, and extremely responsive low input lag on Windows PC!
