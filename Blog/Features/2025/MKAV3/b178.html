UPDATED Mario Kart 64 Amped Up V3 Released

Get Revved Up! ~ Jul 27, 2025

Remodelled Racers

Following the fantastic F3 presentation from Kaze Emanuar, Litronom has released his entry to the showcase, the V3 of his "Amped Up" mod for Mario Kart 64!

The notable changes in this edition include 16 brand-new courses, 5 additional game modes, fully animated 3D racers, modern visuals, groovy music, customisation settings, unlockables, and much, much more!

Patching Practices

In order to play this ROM hack, all you need to do is patch the vanilla Mario Kart 64 (U) [!].z64 ROM (MD5: 3A67D9986F54EB282924fCA4CD5F6DFF) with the provided .bps file linked in the YouTube video, and you can use Marc Robledo's fantastic online patcher HERE to do so.

The resulting 21 MB ROM has an MD5 of D8D3E51FC3419C721703DBD37429A92D, and having patched mine, I'm going to be giving this one a massive play through, and I'll be reviewing it as soon as I'm done, so keep 'em peeled for that!

UPDATE: 29/07/2025 - V3.01 hotfix has been released with the following updates:

-Fixed turning off Modern Visuals in the Title Screen menu while having 3D Racers enabled causing glitched racers.

-Fixed zoom function in classic camera mode.

-Fixed a crash on console due to sound portamento miscalculation.

-Changed ranks updating even if player 1 is currently on Lakitus rod.

-Changed the speedrun verification text to be less intrusive.

-Changed custom menu navigation from internal "menu controller" to controller port 1, in order to allow control on certain systems.

-Changed Boo Stealing option defaulting to "Everyone" and Camera Mode defaulting to "Zoom+" on a fresh save file or on file erase.
