UPDATED Mario Kart 64 Amped Up V3 Released

Get Revved Up! ~ Jul 27, 2025

Remodelled Racers

Following the fantastic F3 presentation from Kaze Emanuar, Litronom has released his entry to the showcase, the V3 of his "Amped Up" mod for Mario Kart 64!

The notable changes in this edition include 16 brand-new courses, 5 additional game modes, fully animated 3D racers, modern visuals, groovy music, customisation settings, unlockables, and much, much more!

UPDATE: 29/07/2025 - V3.01 hotfix has been released with the following updates:

-Fixed turning off Modern Visuals in the Title Screen menu while having 3D Racers enabled causing glitched racers.

-Fixed zoom function in classic camera mode.

-Fixed a crash on console due to sound portamento miscalculation.

-Changed ranks updating even if player 1 is currently on Lakitus rod.

-Changed the speedrun verification text to be less intrusive.

-Changed custom menu navigation from internal "menu controller" to controller port 1, in order to allow control on certain systems.

-Changed Boo Stealing option defaulting to "Everyone" and Camera Mode defaulting to "Zoom+" on a fresh save file or on file erase.

3.0.1 patched ROM MD5: 89700F3B79DC2926535FA071ECDB3EC0

UPDATE: 12/08/2025 - V3.02 hotfix has been released with the following updates:

-Changed the ROM to use the Advanced Homebrew ROM Header, which will automatically set the 16k EEPROM Save Type on supported systems.

-Implemented a potential fix for racers not being properly eliminated in Elimination Mode.

-Implemented a potential fix for racers not properly losing a balloon in Balloon Race Mode.

-Added a notice that trophies are not enabled in classic cups.

-Updated 3D racer models with proper decal sorting to appear less flickery on low-resolution systems/consoles.

-Updated trophy models with proper eye textures.

-Updated the particle system to properly animate particles at 60 FPS.

-Fixed shells currently held by CPU racers so they are properly destroyed when colliding with items or other racers.

-Fixed stray shells on the road after the CPU holding them finished the race through various means.

-Fixed the thunder sound effect on KiKi KaiKai Shrine.

-Fixed Air Gust objects (3DS NBC: Clown Car, SMW Raceway R/T: Pipe) so they properly detect racers that would have overshot in the previous version.

-Fixed Allow Item Usage so Boo and Starman items can retrigger while already active, prolonging and mixing their effects.

-Fixed an issue where racers could pass through the road on some occasions while in first-person camera mode.

-Fixed the honking delay being too short at 60 FPS.

-Disallowed honking for frogs.

-Some larger and smaller tweaks and improvements to the code.
