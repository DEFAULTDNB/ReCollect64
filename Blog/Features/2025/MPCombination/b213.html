Multicart Multiplayer

Over the last couple of years I have become intrigued by compilation carts that sketchy third-party sellers put together on eBay. You know the sort. The 4-in-1 SNES Street Fighter II Collection, or the N64 Turok Collection, which features Turoks 1-3 plus Rage Wars all on one cart. I even spoke to BennVenn about it regarding how multicarts worked and observed his efforts to make a JS-based tool to facilitate the construction of them; sadly, not much progress has been made, and I haven't seen any of these combination carts dumped anywhere for analysis yet.

I point-blank refuse to purchase a multicart because, first and foremost, these are priced at $49.99/Â£40 plus shipping and taxes, and there is no chance I want to outlay that much in an attempt to decompile the methods in which these have been put together, and secondly, these same, unscrupulous sellers are also selling freely available ROM hacks, and the creative tallent behind them sees absolutely zero money of the profit from these sellers' "side hussles".

The bottom line is that any true N64 enthusiast and follower of the scene should definitely not buy into this pirate ecosystem; it's simply not fair on hard-working people like Kurko, NFL Blitz Resurrection, Kaze Emanuar, Litronom, GoldenEye Mods, or Rovertronic, for example, to have their work taken and be given nothing in return.

This weekend though, I stumbled upon something interesting. Something I had not seen before, and something which could prove to be extremely useful for people to create their own compilation carts if it can be modified. That something is the Mario Party Combination hack by Rainchus!

The combination pack aims to simply combine either the first two or all three, Mario Party games into one ROM for use on flash carts or reproduction boards. The purpose is purely to make the experience more streamlined and portable for those with a penchant for the manic board game genre!

If you would like to try this out for yourself, and you happen to have a 96 MB (100,663,296 bytes) rewritable cartridge spare and an expansion pak inserted, you will require:

  • Mario Party 1 (U).z64 MD5: 8BC2712139FBF0C56C8EA835802C52DC (32 MB)
  • Mario Party 2 (U).z64 MD5: 04840612A35ECE222AFDB2DFBF926409 (32 MB)
  • Mario Party 3 (U).z64 MD5: 76A8BBC81BC2060EC99C9645867237CC (32 MB)

Running the patcher will automagically combine the ROMs into one uber-ROM, and depending on which ROMs you include, you will get these files as the output:

MP3-MP2.z64 MD5: 0A77690FBAE955587ACDB28A58FCB3C3 (64 MB)

MP3-MP2-MP1.z64 MD5: 0CE4B7203ADD1EBE62989318F96C5C1E (96 MB)

If you don't have a flash cart, you can always run these compilations in an emulator; just be sure to enable the 8 MB expansion pak and add a 16 kbit EEPROM save type to your save config!

Technical Twist

The included method for loading each game is to press either L, R or Z to select your chosen game. The combination pack also allows you to toggle on/off the 190 mini games in a separate menu too, so there is much more added to this than meets the eye. Rainchus has put in a lot of effort to make this as stable as possible, and version 0.48 from December 2024 is the latest and greatest there is right now.

Because the patcher is hash-based, you would need to make it generic and recompile it, modifying the relevant lines of code to accept any other ROMs instead of the hard-coded Mario Party ones. You would also need to remove all the other hard work that has been put into making this patcher improve the original games, incorporating the work of the current decompilation and recompilation projects.

The best course of action would be to simply ask Rainchus if they could make a version that accepts any 3 or more ROMs, without hash checks. Adding a simple GUI to configure things like a background image and the ability to rename the ROMs and assign them buttons to look neatly presented once loaded on a flashcart, this thing could be universally useful.

Having looked around Rainchus' GitHub pages, I noticed several interesting recompilation projects in the works, as well as something pretty cool: the first working version of the Japanese Chameleon Twist recompilation, based on N64: Recompiled. There is also a tentative first release available there, which allows you to try out the humble beginnings of the statically recompiled version of Chameleon Twist today!
