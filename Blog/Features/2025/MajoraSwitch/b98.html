Majoras Mask Switch Port

Portable Termina! ~ Mar 7, 2025
MM Switch Port Released

Trouble-free Transformations

Using the latest Satoko Charlie 1.1.2 build of 2Ship 2Harkinian, it is now possible to play the Majora's Mask decompilation on your Nintendo Switch!

The port, brought to fruition by CrystalYoshi over at GBAtemp, allows fans of the 2000 time-crunching Zelda game to experience the game, even with mods, on the move.

The instructions are relatively straight-forward, just download and build on a computer, transfer to an SD card and use a modded Switch console to enjoy!

 

Requirements:

+ Original copy of Majora's Mask (NTSC-U N64 OR NTSC-U GC)
+ Modded Nintendo Switch
+ A device to make the mm.o2r file (Windows, Mac, Linux)

Installation:

Download this 2Ship 2Harkinian build on a PC, scroll down and select the platform you are on.
Use the program called 2ship to generate a mm.o2r file by specifying the ROM you want to use.
Download the latest switch port from here.
Make a folder inside the switch folder on your SD card, you'll be storing the nro there.
Copy the o2r file you generated into the folder you made earlier among with the o2r you generated on the last step.

Masters of the Harbour

Built off the back of the Harbour Masters' work with The Legend of Zelda Majora's Mask decompilation effort, this version is the first to publicly allow Switch users to play the game without resorting to NSO or Retroarch emulation.

Upcoming builds will likely feature support for the Japanese version, the European ROM with its multiple languages, additional tweaks to performance and settings, and fixes for running via the HB launcher with a proper icon.

To avoid crashes, please ensure thatvyou are using the latest 2Ship dev build to generate the O2R files. They can be found here. Additionally, if you have trouble mapping buttons, try holding A before entering the button you wish to assign.

It can't be too much longer until the fantastic Star Fox decompilation port Centauri Alfa makes its unofficial way to the Nintendo Switch too, bettering Nintendo's line-up of N64 games on the NSO service that sadly suffers various emulation and lag-related issues.
