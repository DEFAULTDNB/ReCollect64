Mario Kart 64 Decompiled

A Speedy Decomp

Mario Kart 64 has become the next in a series of Nintendo 64 ROMs to be fully decompiled. The progress made within the repository suggests that the USA and EUR ROMs are now 100% matching with known dumps, though there is no update on the JPN version of the game.

With the game fully decompiled, it's only a matter of time until players will get the opportunity to replay the beloved childhood classic with tweaked framerates, enhanced resolutions and a modern control scheme (including gyro controls) in the not-so-distant future, thanks to Mr Wise-Guys' recent N64 recomp framework.

Modern Vintage Gamer initially attempted to build his own version of Mario Kart 64 by using Nerrel's original information, the decompilation project examples and by converting the ROM into a binary *.ELF format. He manually mapped out the entry points, list of functions and OS calls to the *.TOML file required, ready to be fed into the recompilation pipeline. Notably the game had 28475 symbols, 20376 functions and an entry point marked at 0x8000450, but in order for it to be recompiled, the C code needs to be fed into a compiler such as Visual Studio and finally hooked into a renderer such as Dario Samo's RT64.

Cutting out the step of converting the code to C in the as-yet-incomplete/unreleased N64: Recompiled tools allows any decompiled project to be fed into this pipeline ready to be ported to PC or other systems. The irony is that the N64Recomp tools should allow almost instant static recompilation, whereas using the files from any existing decompilation means that years have passed and many hours of effort poured into each one since the projects were first initiated.

Slow & Steady

Getting the project to work as a playable PC port will not be instantaneous. It might yet take weeks or months for the game to be moulded into a playable format that the developers are happy to release. Let us not forget that the speed of a project is dictated by the ability of people, or groups of people, along with their time and energy. Nothing is ever guaranteed and no developer is obligated to be release publicly.

Judging by previous efforts such as GoldenEye 007 the games code was decompiled around 5 years ago, recompilation work began publicly in March 2025 by Kholdfuzion. Just one month later, in April 2025, the game was seen in a 240 FPS playable format, at least for developers, though it has not been released to the public as of yet.

There is already chatter of people wanting to modify this for online play, and adding additional texture/characters as well as tracks and items. With a store like Thunderstore.io linking hosted mods for recompiled projects, this could become an exciting reality.
