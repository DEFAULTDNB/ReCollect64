Marios Modules Update WIP!

New Modules! ~ Nov 13, 2025

Utility Upgrades

Following the rather brilliant release of Rovertronic's Mario's Modules ROM hack back in February, the developer has shared some seriously cool-looking upcoming features this month for the next iteration of his brain-bending game.

Though nothing has been released publicly yet, Rovert has demonstrated a handful of brand-new abilities and combination methods to give the game a fresh QOL update. Check out this quick compilation of his most recent activity:

Changed Components

The rotate module, randomised chests, reroll system, vanity sockets and sequencing modules are just some of the features implemented which will grant the game even more longevity, but I have no doubt that Rovert has even more more genius coding trickery up his sleeve. Between himself and Kaze, almost anything is possible in Super Mario 64 on the N64 these days, so I am very much looking forward to experiencing this update when it lands.

Be on the look out for the new update patch in the coming weeks/months, though there is no set date as to when exactly it will release. Don't forget that you can play the original version of Mario's Modules (v1.10) today by grabbing Rovert's patch from Romhacking.com and using Marc Robledo's online JS patcher.
