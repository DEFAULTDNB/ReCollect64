SF64:Recompiled Released!

"You've Become so Strong, Fox."

Following the recent Dreamcast port and release of Star Fox 64, the Nintendo Switch port, and the prior PC port known as Starship, Sonic Dreamcaster, lead developer of both the decompilation project and Starship (the other PC port) is back with a fantastic Christmas gift by way of Star Fox 64: Recompiled, a version of Star Fox 64 that directly harnesses the ongoing work of Wiseguy's N64:Recompiled, and DarioSamo's RT64 renderer.

 

Key differences with Starship:

* While Starship was made using LibultraShip by Kenix3, the same library that was used for the Ship of Harkinian (Ocarina of time PC Port), Starfox 64: Recompiled was made using N64Recomp by Wiseguy, the same technology used to make Zelda64: Recompiled.

* Uses RT64 by DarioSamo as it's graphics renderer which is the most accurate N64 HLE renderer out there, this means more and better visual effects, automatic texture scrolling interpolation, better matrix interpolation and performance.

* Supports code mods out of the box, the difference is that code mods can contain behaviour changes to the game itself, such as a randomizer, by just copying the mod files inside the mods folder.

* Perfect Audio accuracy.

Starfox 64: Recompiled is compatible with Microsoft Windows, Linux (both ARM and x86) and MacOS.

(The N64:Recompiled project has produced one of the best ways to enjoy these Triple-A titles to date)

Lylat Lineage

In addition to this brilliant release comes Fox Lore, a descriptive timeline of events that documents the history of the project, through the decompilation stages, porting, recompilation and eventual releases that we end users get to enjoy just in time for Christmas.

If you would like to learn about the project's history in granular depth, then please take a read through the Fox Lore documents, which contain some fascinating information, interesting anecdotes, and the exacting timeline of events, including the contributions of additional developers who joined the team along the way.

To enjoy Star Fox 64: Recompiled, you will need to dump or obtain your own Star Fox 64 USA REV1 *.Z64 ROM (MD5: 741A94EEE093C4C8684E66B89F8685E8) and then you can sit back and save the Lylat system.
