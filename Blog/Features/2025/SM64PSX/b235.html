SM64 PSX Port WIP!

32-bit Dip

Since the Super Mario 64 ROM was decompiled in 2019, there have been countless ports to other consoles, but one console has always been neglected, and that's the PlayStation One. It sounds impossible, but yes a Nintendo 64 game will technically be playable on a PlayStation One console!

Well, as of a couple of days ago, developer Malucard has put together an early WIP of the SM64 port for the PlayStation 1, the Sony console that is synonymous with 32-bit wobbling pixels, as opposed to Nintendo's 64-bit true 3D console!

(Image Credit: VideoGameEsoterica)

One Mario Sixty More

Patching the SM64 USA Z64 ROM with the latest BPS on the GitHub repo, you get a 392 MB file that can be loaded into any PSX emulator. The game is missing things like the iconic 3D Mario head, and you're presented with a level select instead of the intro menus, but it's a start!

Graphically, this port is surprisingly adequate, with the polygonal look working quite well, but things like lighting, clipping and culling cause slight visual issues. While the music works and the game is playable to a point, this is still a very early port, and there is still lots of work to do before you could consider this complete in any way.

Keep your eyes peeled for future updates, because this is such an interesting project that finally brings closure to those PS fans who always enquired about getting Super Mario 64 on their PS1s on various forums.
