ROM Patcher JS Updated

Perfect Patching

Marc Max, aka Marc Robledo, updated his super convenient JavaScript-based ROM patcher to version 3.2, which adds support for *.EBP format and allows metadata information to be added to *.RUP and *.EBP formats while creating patches.

While it doesn't mean much for the average N64 flashcart user (unless you're into Sodium64!), it does mean that the 12-year-old *.EBP, or EarthBound Patch format by Lyrositor, which is an extension of *.IPS format, a method of storing differences between two binary files, is now fully incorporated into the application.

Supported Formats:

  • IPS
  • UPS
  • APS (N64/GBA)
  • BPS
  • EBP
  • RUP
  • PPF
  • Paper Mario Star Rod (.mod)
  • VCDiff (.xdelta, .vcdiff)

Core Features:

  • Can patch and create patches.
  • Shows ROM CRC32, MD5 and SHA-1 before patching.
  • Can remove headers before patching.
  • Unzips files automatically.
  • Made in Vanilla JS.
  • Can be run in any modern web browser (including mobile) and Node.js.
  • Can be customized and embeded into your website for a custom patcher.

ROM Patcher JS is a tool that I use almost daily to check out the latest N64 ROM hacks, so this addition makes this invaluable tool all the more robust for all users.
