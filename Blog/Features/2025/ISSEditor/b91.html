ISS98 Editor 0.43b Released

On the Ball

In January, Nicola Cusano and Pedro Seidel on the International Superstar Soccer Fans (Nintendo 64) Facebook group released their editor for ISS 2000.

This time, they have started building an editor for ISS98, which aims to include the same functionality as their last project but for the pre-Y2K edition of Konami's incredibly popular football simulation series.

Similarly, this version auto-detects the ROMs region and can automatically input all the players real names from an included pre-built database. Though it recognises the NTSC-J ROM, it does not yet work with Jikkyou World Soccer - World Cup France '98 (Japan).z64.

ROM Revisions

Microsoft.NETCore.App version 8.0.0 or higher is required to run any version of the ISS editor, but this version does not support visual editing of player's faces yet; you can only flip through the options numerically. The editor also automatically patches your existing ROM, so please do not forget to make a backup of the untouched ROM before you start editing anything.

Compatible ROMs:

  • International Superstar Soccer '98 (USA).z64
  • International Superstar Soccer '98 (Europe).z64
  • Jikkyou World Soccer - World Cup France '98 (Japan).z64 (Not supported yet)

With this project in its infancy, the editor is a little bit more limited than the version for ISS2000, with no ability to edit formations or player's physical looks just yet. These features will come when more offsets and byte codes are discovered. They are not entirely portable between ROMs, and most differ from game to game.

I have no doubts that the team will combine their efforts and build a one-for-all solution for editing any International Superstar Soccer title for the N64 in the not-so-distant future.

#### ISS98 v0.61 Release #### 02/03/2025

What's new and fix:

-Fixed some Real Names wrong

-Fixed Real Names import bug with All stars team. Now automatically refresh with real names.

-Fixed a lot of bugs.

A lot of new features:

-Added the "Transfer" button to move players. Finally, we can fix Ireland lineup

-Each ROM, in its final section, contains empty parts that do not affect its functionality. I used this space to create new players, who will then be transferable to the game's teams. This way, we can create usable reserves.

-Implemented import/export functions for players.

-Added photos for each player present in every version, plus some test photos. If you wish, you can place images in the "photos" folder, which will be loaded into the editor. The image name is determined by the three appearance parameters. For example, for Ronaldo 61-F-B8, the name will be 610FB8.jpg

-Added "COPY/PASTE" buttons to copy appearance between players.

-Introduced STADIUM selection. Now, it is possible to set a different stadium for each team. In the tests performed, no problems were found in the International Cup and World League modes. The team will use the specified stadium.

IMPORTANT: MODIFYING THE STADIUM AFFECTS MEMPAK USAGE. On RMG, simply set "Settings" -> "Core" -> "Save filename format" -> "Internal ROM name" to use the Mempak without issues, if you don't edit ROM name, obviously. If you edit ROM name remember to rename MPK file at the same. I don't use other emulators, so I can't provide instructions for those. Personally, I use savestates, also because any name modification would need to be reset in the game's internal editor. For All Stars Teams, there is the code, so the Mempak is, for me, not necessary.

To avoid this workaround you must don't edit Stadium.

IMPORTANT 2: This version is in beta. Always make backups.
