ISS2K Graphics Tool Released

Custom Kits

A couple of days ago, Nicola Cusano of ISS 2000 25th Anniversary edition fame, as well as the ISS 98 and ISS 2000 editors, released a tool which can easily edit and adjust your team's kit with a bit of imagination and just a few clicks.

The idea is simple: unpack the graphics from the International Superstar Soccer 2000 En/De Z64 ROM (MD5: 9F101B6F6BEF4F267DEB5C6C37A24B97), then edit each panel as you wish, and finally export the modified ROM with your very own customised kit!

 

International Superstar Soccer 2000 – Graphics Tool

Alpha Version by Nicola Cusano

This tool is designed to modify the graphics of ISS 2000 Europe En/De (z64 format).


Release Notes:

– Fixed the issue that corrupted the ROM when reimporting graphics.

– You can now edit graphics freely without risks.

– The ROM is expanded to 32MB instead of the usual 16MB. On modern emulators this causes no problems.

– On older emulators it *might* cause issues; EverDrive and similar devices cannot be tested, but in theory there should be no incompatibilities.


Tips:

– I strongly recommend starting with a clean ROM to understand how the Editor works. If you plan to work on an already modified ROM, make a backup first.


How to use:

– Press **"Extract Graphics Files"**. Select a ROM. All graphic files will be extracted and saved inside the "extracted" folder.

If you already exported once, the tool will ask whether to overwrite the existing files or work on the ones already in the folder.

– To reimport your modified files, press **"Patch ROM!"**.

Kits Editor

**************************


Note: As of today, I have not found the method to assign the primary kit color.

So if you edit Brazil Home, you must always make the shirt yellow.

For the Away kit, the only risk is ending up with the same color as the Referee.


This tool edits kits and their color palettes, letting you work on the 2D texture while immediately previewing the result on the 3D model.

Here’s how to use the editor.


1. Basic Concept: Colors and Indexes

The key thing to understand is that you’re **not** coloring the texture directly.

The system works with indexes (0 to 15).

On the left you have a **Grayscale Palette**: each gray tone corresponds to a number.

You draw using these "numbers".

The **Color Palettes** then decide that “index 1 = red”, “index 2 = white”, and so on.


Quick commands:

Left Click: Paint on the texture.

Right Click: Pick color (Eyedropper).

Grid: Enable "Show Grid" to see boundaries of each part (shirt, shorts, socks). Do not draw in the areas marked "DON’T TOUCH".


2. Color Editing (Palette)

In the center you find the color strips for each kit part (Shirt, Shorts, Socks, etc.).

Change color: Left click on a square → choose color → OK.

Copy color: Right click on a square to copy its RGB values.

Color Preview: Enable this to see the left texture in full color instead of grayscale. Useful for understanding what you’re doing.

Number palette with Preview: Use only colors from 2 to 6 — those are the only ones truly used by the palette logic.


3. Gradient Tool

To avoid drawing shading pixel-by-pixel:

Click **Gradient** above the palette you want to edit.

Set **Start Index** (which square to begin from) and **Steps** (how many squares to fill).

Choose a Base Color and use the **Darkness** slider to determine how much it should darken.

Press **Apply**. The tool will calculate all intermediate colors.


4. Zoom and Details

For working on logos or precise details:

Select the area from the dropdown on the top-right (e.g., KIT: 1 or SL: LEFT).

Press **Open Zoom**.

A dedicated window will open. Anything you draw here is instantly reflected on the main texture and the 3D model.


5. 3D Preview

The model on the right updates in real time with every pixel you change.

Hold the left mouse button and drag to rotate the model.


6. Sleeve Management

Left Short → Right: Clones the left short sleeve to the right.

Left Long → Right: Clones the left long sleeve to the right.


7. Saving and Export

Save: Overwrites the game’s .bin files directly. Save often.

Export Kit: Creates a .kit backup file containing both graphics and colors. Useful for sharing or backing up your work.

Copy Kit from…: If you want to start from another team’s base, select it from the top menu and press Copy.

Transparency note: Usually the first color (Index 0) or the last (Index 15) is used by the game as transparency.

If you see “holes” in the 3D model, check that you didn’t use that index to paint solid parts.


**************************

Intro Screen Export/Import

**************************

As the name says, this extracts the file **Intro_screen.jpg** into the “extracted” folder.

You can edit it freely with any graphic editor. Just keep the original filename for correct reimport.

**************************

Other Graphics

**************************

Works like the Intro Screen: it extracts the selected graphic piece and allows reimport.

Do not change filenames if you want the import to work.

I separated it from the Intro graphic because these are grayscale graphics — in some cases (e.g., player cursor) they are colorized by palettes still not found or mapped in the editor.
