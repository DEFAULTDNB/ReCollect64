Updates from Mr-Wiseguy & Dario Samo ~ May 5, 2025
Zelda 64 Recomp 1.2 Released

Refreshed Recompilation

Almost one year after Mr Wiseguy's initial announcement of N64 Recomp, and Zelda 64 Recompiled, the latest version, 1.2, is now available. This update bring compatibility with Mac OS, texture packs, performance improvements and full modding support!

 

Wondering about your save data when upgrading? Don't worry! Your save data is stored separately to the game itself. You can overwrite your old version and your save data will still be there, completely intact, and will work on the new version.

Modding support has been added. Mods can be installed by dragging them onto the game window or by using the "Install Mods" button in the mod menu before starting the game. You can find many mods on the Thunderstore page at https://thunderstore.io/c/zelda-64-recompiled/. More info can be found in the readme.

Texture pack support has been added. Texture packs are managed with the same menu as mods and installed in the same way. Some texture packs can be found on the thunderstore page linked previously.

MacOS is now supported via a Metal renderer. Thanks @dcvz and @tanmaysachan!

Many graphical optimizations have been added to improve performance on lower end GPUs.

Various UI tweaks and improvements.

A flatpak has been added for the Linux version. Please note that the AppImage has been removed due to some file picker issues it had.

As pointed out by Nerrel recently, the reason we haven't seen lots of static recompilation projects come to light, nor tools, since the initial announcement is due to the ongoing effort being put into refining the framework, behind the scenes, to enable parity across future recompilations and more refined feature-rich experiences going forward.

The project has seen massive performance gains since conception, and now using just a GT 730 with 2 GB of VRAM, you can achieve fully retextured games that run at a stable 60 FPS!

To date, we know that Super Smash Bros., Kirby 64 and the Crystal Shards, Chameleon Twist, Quest 64 and Dinosaur Planet are currently in the works from various people, as well as the previously mentioned Superman 64, Rocket Robot on Wheels, and Banjo-Kazooie from Wiseguy himself. Interestingly, potentially any ROM hacks, for example TLOZ: Indigo, could also be recompiled into 4K, 240 FPS, ultra-widescreen, modern-controlled versions with texture enhancements, mods and more!

Mods and More

The functionality added to this project is essentially a framework for all projects going forward. This means that any games that are given the recompilation treatment will have immediate access to mods, texture packs, cheats, and more.

Thunderstore.io is the latest name in mods, and will deliver patches and features to all games going forward. It already has a variety of content available for the Zelda Recompiled project that you can utilise today. Featuring anything from replacement character models, camera tweaks, and controller overhauls, this store holds everything you could need to improve or customise your game as you wish.

Dario Samo has also hinted that since this framework is now flourishing, RT will now be focused on again, bringing yet another layer of polish to these already impressive projects.

Let's hope that it isn't another year before we can get our hands on the full toolchain to make our own recomps!
