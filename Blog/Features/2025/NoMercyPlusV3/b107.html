Patch V3 notably works on all flashcarts (with an expansion pak installed) as well as most emulators, and it disables anti-aliasing to improve the framerates of four-player matches. Check it out below:

V3 brings the patch to a point where it is open for the public to check out and leave feedback over on Retro Randy's Patreon.

To play the game, you will need to patch the original WWF No Mercy (U) [!] Z64 ROM with an MD5 of 04C492BE7F89FC6F425238BD67629544 using an online ROM patching site or the included DeltaPatcher executable. The download includes the .FLA save file required for each to unlock the full roster, plus a readme to ensure you have the correct save type configured on your flashcarts.

The use of a save game also, brilliantly, prevents unauthorised cartridge production sales. The .FLA validates the game and stops scumbag cloners from making money off these projects, of which none filters back to the original creators!

Your patched ROM should have an MD5 of 37F84949154BBCE3488F4CE7CDE794E6 for the regular patch and an MD5 of B07C8529A49E047DFDEEC9BE5473651E if you choose to patch it for SummerCart64 without save validation. You can also use the latter on the included mupen64plus_next_libretro_over64mb core for RetroArch to achieve the best compatibility on a PC.
