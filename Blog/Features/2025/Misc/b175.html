The Harbour Masters Deliver ~ Jul 23, 2025
OOT & MM Port Updates

Today, the Harbour Masters have updated their The Legend of Zelda: Ocarina of Time & The Legend of Zelda: Majora's Mask PC/Mac/Linux ports, and today's updates give the games more stability, tweaks, and accuracy than ever before.

Shipwright Blair Echo

With a slew of logic and specific scenario fixes as well as some nice improvements tweaking the flow of the game, the full changelog can be found here:

 

Blair Echo (9.0.4)

July 21, 2025

Fixes:

+ Fixed false positives in the rando file version flow.

Blair Delta (9.0.3)

July 21, 2025

Features:

+ Added entrance shuffle to check availability.

+ Removed all support for previous version randomizer files. The files will be renamed and the user notified if present on startup.

+ Also notifies user when trying to use a spoiler log from and old version and prevents parsing.

+ Skip Feeding Jabu is now an option for rando.

Fixes:

+ Fixed numerous logic bugs.

+ Fixed several crashes related to randomizer and rando windows.

+ Fixed some ImGui id overlaps.

+ Fixed C-Down positioning with left-side anchor setting.

+ Fixed issues with Dark Link, especially useful in enemy rando.

+ Fixed dpad navigation on Quest Status screen.

+ Fixed cutscene skip setting changes not applying using All or None.

+ Fixed club moblins moving out of bounds, removed from no-clear-room list.

+ Fixed several typos.

+ Fixed several seed-bleed bugs.

+ Fixed LACS options not having a tooltip.

+ Fixed Ganon's Tower Barrier not dispelling when cutscene is skipped.

Tweaks:

+ Improved the background opacity of docked Item Tracker windows.

+ Various hint improvements.

+ Check Tracker now properly hides excluded locations.

+ Ivan the Fairy can pick up Deku Seeds.

+ Added check for OneDrive path to prevent running and inform user to move install.

+ Added cutscene skips:

+ Water Temple Dragon Statue

+ Kakariko and Hyrule Castle gates

+ Trial barrier dispel

+ Several smaller improvements to hints.

+ Added some hint translations.
