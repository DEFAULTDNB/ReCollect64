Starship Updated

V2.0: Barnard Alfa! ~ May 26, 2025

Port Progression

In an update to the initial release of the Harbour Masters' PC and Nintendo Switch port of Star Fox 64, known as Starship Centauri Alfa, a fresh update, named Starship Barnard Alfa, has just landed that updates and adds a slew of features, brings the port to Linux and will be arriving on macOS soon too.

The information was shared via SonicDcer, who affirmed that they had taken extra care to ensure that this port is as accurate as possible.

 

New Features & Improvements:

Added built-in O2R asset file generation to the game.

Added US 1.0 ROM Support.

Added JP 1.0 ROM Support (JP Voices only).

Added EU 1.0 ROM Support (Lylat Voices only).

Added event hook system, making it easier for devs to mod the game.

Added option to invert the analog stick Y axis.

Added Cheat: infinite boost/brake.

Added Cheat: Laser Range multiplier.

Added Cheat: Rapid-fire.

Improved Jump to Map Developer functionality (Instant).

Added the Beta Coin replacement for the Gold Rings found inside the game.

Added the Beta boost/brake HUD found inside the game.

Added HUD aspect ratio customization independently from game aspect ratio.

Added option to disable Gorgon's boss screen flashes (Accessibility).

Added O2R (ZIP) support for assets, for ease of modding.

Added developer functionality to disable Gamma Boost.

Improved controller compatibility.

Added a more appropriate default controller mapping to the game.

Added Developer Pause.

Added SSE optimizations for Sound.

Added option to show outline for ships in radar.

Added option to show red background for all enemy radio messages (instead of just Andross).

Added option to customize cockpit glass opacity in First Person view.

Improved Matrix Float Precision.

Added 5.1 surround channel sound.

Ending can now be skipped if it was seen at least once.

Added texture interpolation for planet floors.

Added Render parallelization

Bugfixes:

Improved stability.

Fixed All-Range ground graphical bugs.

Fixed linux build.

Fixed collision bugs in Meteo and potentially other places.

Fixed some sound effects not playing correctly.

Fixed Venom 1 Boss "Golemech" electricity effect.

Fixed medals in the ranking.

Fixed data in the default ranking.

Fixed incorrect texture displacement for the radio noise.

Fixed items not despawning behind the player in All-Range mode.

Fixed fullscreen toggle bind key message.

Fixed alignment of the training ring counter.

Fixed texture scroll for the train wheels in Macbeth.

Fixed Sector Z actor spawner and added an option to correct a vanilla bug.

Fixed Titania's Boss "Goras" graphic issues.

Fixed audio crackling on Mac & Linux.

Fixed Zoness water shadows.

Fixed Developer Checkpoints to work per-level.

Fixed more graphic draw calls to widescreen.

Fixed Stereo audio phase shift.

Fixed muzzle flash interpolation.

Fixed Invulnerable cheat for solar and terrain.

Fixed some cutscenes to play at N64 Hardware speed.

Fixed bug where the ending wouldn't play after one sequential playthrough.

Fixed Electric Arc interpolation.

Fixed Mountain graphics in Corneria & Fortuna.

Fixed white flash that happened during Sector Y intro cutscene.

Polished Project

Crucially, this update allows all V1.0 ROMs to be used (PAL/NTSC-U/NTSC-J), OTR modding has been simplified, and it adds 5.1 surround sound output to the mix as well.

The various bug fixes also bring a fresh lick of polish to an already stellar project, and a number of small issues such as audio crackling, cutscene speed, shadows and collisions have now been painstakingly adjusted for parity with the original game.

Thanks to the team of developers, Starship is easily the best way to enjoy Star Fox 64/Lylat Wars on a variety of modern platforms and hardware, so get downloading and start saving the Lylat system today!
