Duke Nukem: ZH Decompiled

Duke's First Decomp

On Friday, BlueSky user Gentarkin shared with me some brilliant news that Duke Nukem: Zero Hour had been 100% decompiled, and what's more, the code is editable for use directly on consoles.

The decompilation has taken Gilles Siberlin and Rainchus around three years to complete, according to the repository, but the resulting code is a 100% match to that of the US ROM, with preparatory efforts being made to ensure that the French PAL version of the game will also see a decompilation too.

 

Hey, thought I'd pass along the message that Duke Nukem: Zero Hour now has a 100% matching decompilation. It's even shiftable, which means it can be modified and recompiled and still work on console.

Duke Nukem: Zero Hour released on August 31, 1999 in the Americas, and on September 24, 1999 in Europe. I remember the game feeling like an extreme departure from Duke Nukem 3D, but the exploration, action, and multiplayer was quite playable!

Let's Rock!

With any decompiled N64 game comes the inevitable question: will someone pick it up to build a recomp, or will it get lovingly ported to the Dreamcast? The progress of the project means that someone skilled could use this codebase to begin a recompilation project using the tools set out by Wiseguy and Dario, a DC port could spring up from JNMartin84, a Vita port from Rinnegatamante, or a PC port could come to fruition through the Harbour Masters.

Hopefully it will be sooner rather than later that Duke Nukem: Zero Hour gets a fresh outing, because this third-person, time-travelling adventure (with a 4-player FPS deathmatch mode) was relatively well received at release and was only exclusively available on the 64-bit console. A multiplayer game like this would be incredible to play on the Dreamcast, with its four-controller support, and the single-player campaign would be great to replay with modern controls, higher frame rates, texture packs, and higher resolutions!
