Ocarina of Time PS Vita Port

Ocarina of Sublime

Developer and porting enthusiast Rinnegatamante has now ported Ship of Harkinian to the PlayStation Vita! You can download the first release, Version 1.0, directly to your console using VitaDB or the VitaDB downloader, or alternatively, you can obtain the VPK here.

The game currently runs at a rock-solid 20 FPS, which doesn't sound like much, but it's more than enough for the 64-bit console, and seeing The Legend of Zelda: Ocarina of Time running on the portable, it looks incredible in motion!

 

Description:

Ship of Harkinian is a sourceport of The Legend of Zelda: Ocarina of Time based off the decompilation efforts of the game.

It features several enhancements and mods support. The Vita port is based on a custom old version (more or less Khan Charlie (6.1.2)).

Requirements:

libshacccg.suprx

Game Data Files: N64

Link to the Source

This port is a fork of the Ship of Harkinian project (Shipwright), which utilises the Zelda Decompilation to produce a working facsimile of the titular action-adventure game.

As with all ports, you'll need a compatible ROM, which can be verified here, and the relevant OTR files extracted from it, because the Ship does not include any copyrighted assets.

 

Download and install soh.vpk and extract the data files in ux0:data.

Download OTRGui.zip from the GitHub page and extract it in the same folder where you have your rom.

Launch OTRGui and generate the required otr file using your rom as source.

Place the otr file in ux0:data/soh.

Don't forget that you can run these files on a hacked PS Vita or Vita TV unit, so you can enjoy this on the go or on your TV screen with full controller support.

Fantastic work yet again by Rinnegatamante, who is continuing to breathe fresh life into the Vita and Nintendo 64 games in 2025!
