WWF No Mercy 97-99 Mod

Retro Rampage

Last week, MFillaz released a rather nostalgic mod pack for Retro Randy Price's WWF No Mercy Plus V3 Beta 5 ROM hack. The pack, dubbed "Pack 1", contains a save file, complete movesets, tag teams and some fun extras.

The author has used VPW Studio to inject textures, giving your favourite muscular faces a boatload of new attires and looks. They have also included the GLide 4.0 plugin to facilitate easier installation on emulators like Project 64 and Mupen64Plus.

 

**Extract contents of 'WWF No Mercy + V3-B5' to your designated Textures folder. See screenshots for example Project 64 folder hierarchy.

**Move/copy 2 save files (.fla and .pj files) to your designated Saves folder.

There are no hacks/Gameshark codes used.

No blood textures as of yet, so if your guy gets busted open, you'll see the modded textures.

Sid copied over Biker Undertaker.

Sgt Slaughter copied over Howard Finkel (so no Cobra Clutch, sorry).

Ho and The Kat left as default.

There are a few unused textures/folders in the file that I may use later.

This is the first mod/pack I've done, so there may be some screwy things here and there, let me know if you see anything weird.

The 97-99 pack also contains classic banners and liveries to spruce up the arenas with familiar 90s events such as Summer Slam, King of the Ring, In Your House, Heat and Survivor Series.

This is a texture modification for the WWF No Mercy Plus V3 Beta 5 ROM hack, and is recommended for use with Project 64 and GLide 4.0 emulator/plugin combination. All you need to do is extract the textures from No Mercy V3 Beta 5, overwrite with the supplied files, and load the save file to gain access to the modified CAWs.

Reaching out to MFillaz, he stated that the pack will "probably be added to over time, at some point, maybe", so if you're a retro WWF fan, keep an eye out for any updates or a "Pack 2" in the future!
