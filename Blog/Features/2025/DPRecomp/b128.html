Dinosaur Planet Recomp Released

Prehistoric Prototypes

Following yesterday's news about the fantastic progress surrounding Zelda 64 Recompiled, Ethan Lafrenais has released their first version of the Dinosaur Planet recompilation.

Dinosaur Planet was set to be an action-adventure game that would rival The Legend of Zelda: Ocarina of Time, which was developed around 1999-2000 by Rare for the Nintendo 64. The title was famously cancelled in 64-bit form, and it was retooled as Star Fox Adventures, Rare's only release for the GameCube, in 2002.

As with all recompilation projects, you will need an unmodified Dinosaur Planet prototype *.Z64 ROM (MD5: 49F7BB346ADE39D1915C22E090FFD748) that was leaked in 2021 by Forrest of Illusion. Please note that patched or "Dino Enhanced" ROMs will not work for this release, so ensure that you have the unmodified "rom.z64" file and not the "rom_crack.z64" ROM. The result of this is that the gameplay is in an extremeley unfinished and unstable state, so for now, you should temper your expectations and expect a lot of crashes until the modified ROM is supported.

Recompilation Reinvigorated

The project, which is based on ZestyDev's decompilation of the leaked Dinosaur Planet ROM, has been worked on in relative secrecy since the game leaked in 2021. The developers have used a method they have dubbed "Precomp" that is essentially using the in-progress decompilation project as a source of symbols for traditional binary hacking.

 

The process involves taking the elf file generated from an in-progress decomp and using it in conjunction with ld to be able to write assembly and C (and C++ for the especially brave) that references symbols from the decomp.

It also allows you to write jump hooks that reference offsets from decomp symbols/segments, allowing you to specify instructions to overwrite without needing to deal with converting between RAM and ROM addresses.

Using ld also ensures that source files are linked against the correct RAM address.

The Dinosaur Planet recomp is based on the as-yet-unreleased Rocket Robot on Wheels recomp known as "RockRet", which was started by Mr-Wiseguy as early as 2020. Using Wiseguy's frameworks, this and many more recompilation projects should soon see the light of day, as previously mentioned in yesterday's article.

Incidentally, the recompilation for Super Smash Bros. is also held on ZestyDev's GitHub pages, which is an exciting project to look forward to, and though there has been a little gameplay footage of Super Smash Bros: Recompiled released by Nerrel, there is currently no public release date for this project.
