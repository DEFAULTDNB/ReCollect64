Mystical Ninja Recomp

Bring Your Own ROM

On July 6th, 2024, developer Klorfmorf gifted the world a pre-release recompiled PC port of the N64 classic, Mystical Ninja Starring Goemon. This was released via their Discord server and touts a launcher similar to the recent Majoras Mask N64Recomp release (interestingly, the main *.exe still uses the Majorask Mask icon), widescreen support, 60 FPS, and bindings for modern controllers.

someone on a goemon discord server I'm in released a recompilation for MNSG, and getting to play the game in widescreen and 60fps is such a blessing (though the recomp only supports the japanese version AFAIK) pic.twitter.com/6KwhkcSZTs

— stoopidmansidiot (@rondleman) July 6, 2024

The news was initially shared by user @Rondleman on Twitter but has since been shared on various news sites along with a link to the initial release executables, which can be found below.

As per all N64Recomp releases, you must supply your own ROM to use this port and keep it legal to redistribute. This time, the ROM required is the Japanese version: Ganbare Goemon Neo Momoyama Bakufu no Odori [J].z64

Preliminary Release

As you can see, the game looks stunning, but it is still lacking a few touches, such as HD textures, a proper widescreen HUD, and various performance/effects fixes before it can be considered a full-fledged release. Regardless, it is hugely encouraging to see more decompiled games getting a modern treatment.

One thing you should bear in mind is that it is currently all presented in Japanese, so it might be a little tougher to play through if you aren't well versed in the language. I'm sure an English version of the Recomp will follow soon enough.

UPDATE: 11/08/2024 - 0.0.2 has been released with audio fixes, graphics fixes, and now supports the english version of the game!
